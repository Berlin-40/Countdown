# Countdown App

**Android · Kotlin · Jetpack Compose**

---

## Description courte
Application Android en Kotlin (Jetpack Compose) permettant de créer, gérer et partager des compte‑à‑rebours visuels pour événements personnels ou professionnels.  
Inclut notifications programmables, thèmes, export/import et affichage plein écran.

---

![image](https://github.com/Berlin-40/Countdown/blob/0acf0f20c74eb6721b539880c779186f63c4fc1c/minuteur.png)

## Fonctionnalités

- **Création de compte‑à‑rebours** : titre, date/heure, description, image optionnelle.  
- **Liste et vue détaillée** : suivi du temps restant en temps réel.  
- **Thèmes et personnalisation** : clair/sombre, choix de couleur par compte.  
- **Notifications locales programmables** : rappels configurables (ex : j‑1, j‑1h).  
- **Mode plein écran** : diffusion des compte‑à‑rebours lors d’événements.  
- **Sauvegarde persistante** : stockage local avec Room.  
- **Widgets et partage** : widget sur l’écran d’accueil (roadmap) et export d’image d’un compte‑à‑rebours.

---

## Technologies utilisées
- **Kotlin**
- **Jetpack Compose**
- **Room** pour la persistance
- **Coroutines** pour les tâches asynchrones
- **Notifications locales Android**
